Projects

My Games List

An application built with the intention of helping gamers with the ever-persistant issue of keeping track of what their vast game libraries, particularly for those who use multiple platforms. Add games to your list and keep them organized between those of which you are currently playing and those which you have completed or have dropped, as well as your very own Wishlist. This application utilizes the IGDB API to quickly validate and obtain specific game information. This application uses a Node and Express Web Server and is backed by a MySQL Database.
